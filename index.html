Digitação em um Site

Toda a parte de digitação usara Funções Como H e P Sendo elas Sediadas dentro de Body

A FUNÇÃO H !!!

...É usada para Títulos e Sub-Títulos sempre ficando no Topo da página
   ...Função H1 (por questões puramente de aparência) é sempre usada para títulos porem as suas subsequentes como h2 ou h3 podem ser usadas para subtítulos
   ... Quanto maior o número da função H menor será a fonte

A FUNÇÂO P !!!

A função P é usada para a descrição dos títulos e Sub-Títulos

Função BR [BREAK] e Espaçamentos de Linhas !!!

A função br [break] é usada para dar espaços entre uma linha e outra
   ... comunmente usada para criar sub-tópicos
   ... Para espaçar mais as lihas é comum de se usar o comando (&nbs p) com todas as letras juntas

Linhas Horizontais [HR] !!!

Para a melhor aparencia De sua Front a função HR adiciona linhas horizontais, úteis para a separação de tópicos

Elemeto de Enfâse !!!

Pode ser usado para mudar a fonte do que estiver entre os dois comandos
   ... (em)
   ... (strong)

Links Em Palavras

Para colocar Link em Palavras se usa a função a
    ...após isso deve-se colocar a referecia do site a qual seremos levado, usanndo a função href
    ...Após deve-se colocar o link entre as aspas e colocar na frente qual a frase que o redicionara para a pagina indicada previamente
    ...Para que o site abra em outra aba se usa a função Target=_blank
